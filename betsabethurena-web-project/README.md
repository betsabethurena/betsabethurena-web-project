# betsabethurena-web-project

## Project 2 - HTML Foundation Page

### Description
This project is a fanpage about the Shadowhunter Chronicles by Cassandra Clare. It presents information about the book series, themes, settings and time periods.

### What I Learned
While creating this page, I learned how to structure a webpage using basic HTML elements such as headings, paragraphs, lists, tables, and links. I also learned how to style a page using HTML attributes without CSS, including colors, alignment, and spacing.

### Challenges
One challenge I faced was working without CSS, which made it harder to control spacing and layout. I also had some issues with file paths when linking pages in different folders, and with changes not appearing due to not saving or browser caching.
